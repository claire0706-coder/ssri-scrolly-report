# SSRI Clinical Report

A static scrollytelling webpage for a 15-minute case-based report on SSRI adverse effects, clinical management, discontinuation syndrome, and hyperbolic tapering.

## Open locally

Open `index.html` directly in a browser.

## Deploy with GitHub Pages

Use GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
